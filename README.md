# Isca analysis

A project for analysing output from some experiments using the Isca global climate model.

To run code which requires them, one would need to obtain GPCP and CERES data from the sources below,
specifically CERES_EBAF-TOA_Ed4.2.1 monthly averaged and GPCP 2.3 yearly averaged precipitation climatology.
and add them to the `data/GPCP` and `data/CERES` directories respectively. We thank these projects for their data.
